## MCP Servers and Plugins

| Server | Value | When to use |
|--------|-------|-------------|
| **firecrawl-mcp** | Scrape + web search, clean markdown without HTML parsing, batch operations | 20+ pages at once, web search with filters, download site documentation |
| **scrapegraph-mcp** | LLM-driven structured extraction (JSON) from complex pages via prompt | Extract `{title, price}` from a product page directly into JSON |
| **agentmemory** | Semantic memory across sessions — agent recalls past without manual copy-paste | Accumulate error patterns over a week, find similar case in new session |
| **codegraph** | AST navigation: who calls a function, dependency chain — without grepping the whole repo | What breaks when you change an interface — in seconds, not hours |
| **context7** | Current library documentation into context, version-accurate — no hallucinations | Write code against Supabase JS v2 / Next.js 15 / any fresh lib |
| **computer-use** | Control native desktop apps via mouse/keyboard — the only way for apps without an API | Finder, Maps, Notes, desktop forms — without a browser |
| **mcp__playwright** | Browser with DOM access, clicks, forms, network interception — more reliable than coordinate clicks | Log in, click, screenshot, E2E test |
| **mcp__Figma** | Read tokens, layers, frame screenshots from Figma — precise code without guessing the design | Implement a component 1:1 against a design system |
| **mcp__supabase** | Manage Supabase (schema, migrations, SQL, edge functions, logs) without leaving the chat | Create a table, apply a migration, debug a slow query |

## Selection Policy

Quality comes first. Token cost is only a tie-breaker after fit and expected output quality are effectively equal.

```
value(skill | project, task) = effectiveness × fit × right_size − switch_cost
pick = argmax value ; tie-break → lower cost_tier
```

| work | model |
|------|-------|
| product-critical · hard synthesis · architecture · final pass | **Opus** |
| standard build / analysis / code | **Sonnet** |
| bulk / mechanical: grep, classify, format, sort, leaf-workers | **Haiku** |
