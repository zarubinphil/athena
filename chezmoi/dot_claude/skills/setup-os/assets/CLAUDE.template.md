## Экономия токенов
1. Внешние данные — сначала разведка (PDF `pages:"1-3"`+grep; Firecrawl `limit:3`,`summary`; веб `onlyMainContent`).
2. Пути к файлам — `os.listdir()`+`os.path.join()`, не строкой (NFD-Unicode, U+202F).
3. Большой результат инструмента → в субагент с точным заданием, не серией Read.
4. Проверяй права/путь ДО работы (`ls -la`), не после ошибки.
5. Не плодить мёртвые шаблоны — файл только если его кто-то читает.
6. Сортировщики физически `mv`, не «предлагают». Нераспознанное → `_unsorted/`.
