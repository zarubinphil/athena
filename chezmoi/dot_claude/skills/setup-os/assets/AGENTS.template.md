# AGENTS — реестр вики-роя (Мнемозина)

> Импортится из vault `CLAUDE.md` через `@AGENTS.md`. Один находимый реестр роя.

| Агент | Роль | Модель | Когда запускать |
|---|---|---|---|
| `kb-ingest` | raw → wiki-страницы + кросс-ссылки `[[]]` + индекс + лог | Haiku/Sonnet | новый файл в `raw/` |
| `kb-router` | триггер → раздел по `_ROUTING.md` (≥70%) | Haiku | классификация входящего |
| `kb-synth` | свод / переписать концепт-страницу | **Opus** | спорный синтез, противоречия |
| `kb-health` | противоречия / сироты / пробелы | Sonnet | weekly health-check |

## Правила роя
- Тяжёлое чтение (PDF/веб/картинки) → листовым воркерам, не в основной контекст.
- Каждый агент возвращает выжимку (caveman), не сырьё.
- Дубль существующего скилла vault (агенты роя) — НЕ плодить, ссылаться.
- Frontmatter каждого: `name`, `description` (триггер-first), `tools`, `model`.
