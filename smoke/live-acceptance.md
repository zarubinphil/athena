# Live-приёмка — чистый Mac / VMware-гость

Дженерик-чеклист ручного прогона на чистой macOS. Дополняет автотесты (`smoke.sh` гоняет инварианты, тут — реальный E2E с сетью, паролем Mac, попапами). Нет PII.

## Подготовка гостя
- VMware Fusion → чистый macOS (Sequoia+), новый юзер, **без** Homebrew/node/claude.
- Снапшот `clean` ДО прогона (откат между попытками).
- Сеть есть (curl, github, npm, brew).
- Apple ID для App Store по желанию (часть интеграций может спрашивать).

---

## Путь A — нуб-установка (пункт 2)

### Шаг 0 — preinstall (Терминал, руками)
```sh
curl -fsSL https://raw.githubusercontent.com/zarubinphil/athena/main/preinstall.sh | bash
```
Наблюдать:
- [ ] xcode-select поп-ап появился, скрипт **ждёт** его (не летит дальше), затем продолжил.
- [ ] Homebrew поставился; на Apple Silicon путь `/opt/homebrew`, на Intel `/usr/local` — brew подхватился в текущую сессию.
- [ ] base CLI (git и пр.) на месте.
- [ ] `claude` доступен (`npm i -g @anthropic-ai/claude-code@2.1.183`). **Если версия ротнулась** — баннер npm о новой версии; зафиксировать (пин бампить, см. preinstall.sh).
- [ ] репо склонился в `~/athena`.
- [ ] финал печатает «набери `/setup-os`».

### Шаг 1 — дирижёр
```sh
claude          # авторизоваться
/setup-os
```
Наблюдать:
- [ ] опрос интеграций идёт **попапами** (AskUserQuestion), не простынёй текста.
- [ ] onboarding-grill первого запуска отрабатывает (Карпаты mandatory).
- [ ] bootstrap слои 0→6 + 0b идемпотентны, без хардкод `/Users/...`.
- [ ] `.athena-version` стамп записан.
- [ ] финальный `smoke.sh` зелёный.

### Капчур точек трения (постмортем E9+)
На КАЖДЫЙ затык: `EN | шаг | симптом | детерминир.? | фикс`. Пусто = чисто.

## Путь B — обновление живой системы (пункт 3)

Гость с **предыдущей** версией каркаса (склонить старый коммит ДО релиза, развернуть, снапшот `old`).

```sh
cd ~/athena
git -C ~/.claude ... # система уже живая
claude
# скилл athena-update
```
Наблюдать (`skills/athena-update`):
- [ ] фаза анализ→план→**согласование** (план показан, ждёт добра).
- [ ] бэкап снят ДО изменений.
- [ ] `git pull` — **ff-only** (не merge-коммит, не rebase молча).
- [ ] `chezmoi diff` показан перед apply.
- [ ] smoke после apply зелёный.
- [ ] **откат**: симулировать провал (битый шаблон) → откат к бэкапу отрабатывает, система живая.
- [ ] `.athena-version` обновился.

## Вердикт
- Всё зелёное + ноль детерминированных затыков → **GO релиз**.
- Любой детерминированный затык → постмортем-строка → фикс → повтор со снапшота.
